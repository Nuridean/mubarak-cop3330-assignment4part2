Assignment 4 part 2 GUI application 

*Allows the user to create task items with a description, date, completion status and save the list of tasks to external storage

How to use 

    -Add new item, use the textfield to add a description and use the date picker to chose a date and hit the add button

    -Delete selected item, left click on one of the items in the tableview and then click the delete button

    -Edit item, double click on an items description or date to change it, press enter to confirm the changes

    -Display items, use the two check boxes above the table to filter out which tasks are visible 

    -Saving the list, use the save file button to save the list to external storage
